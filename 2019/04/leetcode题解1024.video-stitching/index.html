[leetcode题解] 1024.Video Stitching

You are given a series of video clips from a sporting event that lasted T seconds. 
 These video clips can be overlapping with each other and have varied lengths.

Each video clip clips[i] is an interval: it starts at time clips[i][0] and ends at time clips[i][1].  
We can cut these clips into segments freely: for example, a clip [0, 7] can be cut into
 segments [0, 1] + [1, 3] + [3, 7].

Return the minimum number of clips needed so that we can cut the clips into 
segments that cover the entire sporting event ([0, T]).  If the task is impossible, return -1.

题意

给定一组视频片段，每个片段都有相应的起止时间，并且可以随意将视频片段进行裁剪，如：片段[0, 9]可以被裁剪为[0, 2], [2, 8], [8, 9].
现在给你一个时间范围[0, T], 要求你通过对这一组视频片段进行裁剪拼接作出一个在[0, T]内的视频片段，并且拼接的片段数量最少。

Example：

Input: clips = [[0,2],[4,6],[8,10],[1,9],[1,5],[5,9]], T = 10
Output: 3
Explanation:
We take the clips [0,2], [8,10], [1,9]; a total of 3 clips. Then, we can reconstruct the sporting event as follows: We cut [1,9] into segments [1,2] + [2,8] + [8,9]. Now we have segments [0,2] + [2,8] + [8,10] which cover the sporting event [0, 10].

Input: clips = [[0,1],[1,2]], T = 5
Output: -1
Explanation:
We can’t cover [0,5] with only [0,1] and [0,2].

题解

  1. 动态规划 设dp[i]表示[0, i]之间视频拼接的最小数量，那么dp[i] = min(dp[k]+1)(m< k < i), m表示所有包含时刻i的视频片段中最早的起始时刻。初始条件dp[0] = 0, C++代码如下：
class Solution {
public:
    int videoStitching(vector<vector<int>>& clips, int T) {
        //对视频片段进行排序，起始时间早的排在前面，起始时间相同的，结束时间早的排在前面
        sort(clips.begin(), clips.end(), [](vector<int>& a, vector<int>&b)-> int {return a[0]<b[0]||(a[0]==b[0]&&a[1]<b[1]);});
        vector<int> dp(T+1, 110);
        dp[0] = 0;
        int j = 0;
        for(int i = 1; i <= T; i++){
            //找到排序后的片段中，第一个包含时刻i的片段，并且该片段也是从时刻i往前能切出最多分片的片段。
            while(j < clips.size() && clips[j][1] < i) j++;
            if(j==clips.size()) break;
            for(int k = clips[j][0]; k < i; k++){
                dp[i] = min(dp[i], dp[k]+1);
            }
        }
        return dp[T]==110?-1:dp[T];
    }
};

int videoStitching(vector<vector<int>>& clips, int T) {
    sort(clips.begin(), clips.end());
    int cnt = 0, cur_end = 0, pre_end = -1;
    for(auto& v:clips){
        if(v[1] <= cur_end) continue;
                if(v[0] > cur_end) return -1;          
        if(v[0] > pre_end){
            pre_end = cur_end;
            cnt++;
        }
        cur_end = v[1];           
        if(cur_end >= T) return cnt;            
    }
    return -1;
}
